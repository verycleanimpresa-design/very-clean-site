## Pubblicazione rapida su GitHub Pages
1. Crea un repository pubblico (es. `very-clean-site`).
2. Carica i file di questa cartella nella root del repo (`index.html`, `css/`, `assets/`, `sitemap.xml`, `robots.txt`).
3. Vai su **Settings → Pages** e seleziona la branch `main` (cartella `/root`). Salva.
4. L’URL sarà `https://verycleanimpresa-design.github.io/very-clean-site/`.
5. (Opzionale) Dominio personalizzato: configura il tuo dominio e attiva **Enforce HTTPS**.

## SEO
- `sitemap.xml` e `robots.txt` sono già impostati con il tuo URL.
- Dopo la pubblicazione, invia la sitemap a Google Search Console.
